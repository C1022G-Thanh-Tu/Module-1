1. JavaScript (viết tắt là JS) là gì?
JavaScript là một ngôn ngữ script dựa trên đối tượng nhằm phát triển các ứng dụng Internet dựa 
trên client và server.

2. Các cách nhúng JS vào HTML?
Viết thẻ <script> dưới thẻ đóng của thẻ <body>
Viết thẻ <scrpit> (trong thẻ <body>) có src link tới file có đuôi .js

3. Muốn chú thích ở JS, thì làm như thế nào?
Có 2 cách:
- // chú thích 1 dòng
- /* */ chú thích nhiều dòng

4. Các hàm dùng để nhập dữ liệu ở JS?
oncharge
onmouseover - onmouseout
onclick

5. Các hàm dùng để xuất dữ liệu ở JS?
alert
console.log()

6. Ý nghĩa của debug và cách thực hiện debug ở Webstorm?
Debug là quá trình tìm kiếm và sửa lỗi trong chương trình => Giúp lập trình viên tạo ra chương 
trình hoàn chỉnh - không có sai sót trước khi chạy chương trình đó cho khách hàng

Cài debugger trong webstorm để thực hiện debug / hoặc thực hiện các thao tác debug khác như đọc mã nguồn,
chèn các lệnh in ra các giá trị trong từng đoạn của chương trình
