1. Khái niệm vòng lặp?
Vòng lặp cho phép tự động thực hiện một khối lệnh lặp đi lặp lại nhiều lần dựa vào một điều kiện cho trước

2. Các loại vòng lặp trong JS?
- Vòng lặp for
- Vòng lặp for-in
- Vòng lặp for-of
- Vòng lặp for-each

3. Các biểu thức và luồng thực thi của for?
Các biểu thức: 
- Initial-action
- Loop-continuation
- Action-after-each-iteration
- Statement(s)
Luồng thực thi: Initial-action => Loop-continuation => Statement(s) => Action-after-each-iteration

4. So sánh giống và khác nhau giữa for, while và do..while?
- Vòng lặp for:
for (lệnh khởi động vòng lặp đầu tiên; điều kiện tiếp diễn; lệnh kết thúc mỗi vòng lặp) {
    // các đoạn mã được thực hiện lặp
}
- Vòng lặp while:
while (điều kiện tiếp diễn) {
    // đoạn mã được thực thi lặp đi lặp lại
}
- Vòng lặp do..while:
do {
    // mã lặp
}
while (điều kiện tiếp diễn);

5. Phân biệt break và continue?
- Break dùng để thoát khỏi vòng lặp mà không đi tới điểm cuối
- Continue để nhảy qua một phần của khối lệnh và bắt đầu dòng tiếp theo
